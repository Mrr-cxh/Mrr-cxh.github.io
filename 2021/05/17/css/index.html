前端基础

# 概览

层叠样式表 (英文全称：Cascading Style Sheets) 是一种用来表现 HTML（标准通用标记语言的一个应用）或 XML（标准通用标记语言的一个子集）等文件样式的计算机语言。CSS 不仅可以静态地修饰网页，还可以配合各种脚本语言动态地对网页各元素进行格式化。

CSS 能够对网页中元素位置的排版进行像素级精确控制，支持几乎所有的字体字号样式，拥有对网页对象和模型样式编辑的能力。

# Css 基础用法

overflow: hidden;
text-overflow: ellipsis;
white-space: nowrap;
单行文本超出，显示省略号,前提是块元素

user-select:none 把双击选中取消

resize: none; 取消文本域之类的可拖拽变大小

appearance:none 相当于消失 input如果用了这个会不显示，但大小还在

background-attachment: fixed;
/* 滚动滚动条的时候，图片随着滚动条滚动 */

css中虚拟宽高 outline,在边框外(不算真正宽高，只是附在边框上，四周有线)
outline: none; /* input框获取焦点时，四周没有边框 */

input:focus{border:1px solid red} input获取焦点时，边框改变颜色
.check{vertical-align:-2px} 设置元素垂直对齐方式，此时checked与提示在一条线居中

li:hover,.default,.default:hover{color:red} 默认选项

nav: display:table (想把nav变成像table一样的排列方法)
ul: display:table-row 行
(上边的这个nav和ul 的案例是奥斯卡导航)
li: display:table-cell 列 (这个不可以加浮动，文字默认左对齐)

vertical-align:定义行内元素的基线相对于该元素所在行的基线的垂直对齐 top,bottom,middle……
letter-spacing:字符间距 word-spacing:英文单词间距 line-height:行高
text-align:center/left/right/justify(两端对齐):文本对齐方式
text-decoration:none,overline,underline,line-through 下划线
text-indent:首行缩进

font:Bold 18px "幼圆"
font-family:字体 font-size:字号 font-style:Italic斜体 font-weight:Bold粗体
solid:实线 dashed:虚线 dotted:点线

当内容溢出时，overflow取何值
hidden:超出部分不可见 scroll:显示滚动条 auto:如有超出部分，显示滚动条
text-overflow:clip/ellipsis:当对象内文本溢出时，裁剪/显示省略号

水平居中:图片，文字:text-align:center div水平居中:margin: 0 auto
将line-height属性值设置为和height属性值一样为垂直对齐

font-size:0 给父级元素添加，可以解决间隙，也可用在<img>上

文档流对象: block,inline,inline-block
display:block,inline,inline-block(和行内元素在一行显示，但宽高可以设置)

浮动定位:浮动元素在排列时，只参考前一个元素位置即可
clear(清除浮动):both,left,right,none
.clearfix:before,.clearfix:after{
       content:"",
       display:block,
       clear:both
}
.clearfix{
       *zoom:1
}
在子元素浮动时，父级元素可能会出现高度不能自适应的情况，所以要清浮动，
页面乱一般都是由于加清浮动

定位:position:fixed/relative/absolute/static(默认，无定位)
fixed:固定定位 相对于浏览器窗口 浏览器左右的广告，但必须要有宽度
relative:相对定位 直接父元素 原有位置不丢失
absolute:绝对定位 相对static定位以外的第一个父元素进行定位 原有位置丢失
绝对定位：原有位置丢失，会摞在一起
绝对定位：top: 0;  left: 0; 这两个要加
解决绝对定位：设置第一个图片为显示(opacity:1或display:block)，其他为隐藏(opacity:0或display:none)
定位只能通过top,right,bottom,left来设置位置
relative通常和absolute结合使用，父元素用relative,子元素用absolute
z-index:谁的取值大谁就在上面
position:sticky 可做导航粘连效果，用top:0 来设置

border-radius:圆角边框 为50%时，是一个圆

box-shadow:边框阴影: h(左) v(下) 模糊度，颜色，h(左) v(上) 模糊度，颜色，内阴影inset
其中为负值时为反方向，不用写outside 否则会出错
text-shadow:文本阴影: h v 阴影距离 模糊度 颜色
vertical-align:middle/top/bottom 垂直对齐
使用雪碧图时，background-position 要用负值，要量取左上角的位置

:first-child 与 :last-child 单独指定第一个子元素和最后一个子元素的样式
:nth-child(n) 与 :nth-last-child(n) 对指定序号的子元素使用样式 从1开始
这两种是：先定位位置，再匹配子元素(类型)
:nth-of-type(n) 与 :nth-last-of-type(n) 先定位类，再定位位置
odd:奇数 even:偶数 (2n):2,4,6…… (2n+1):从第1个开始，每2个循环1次
例：h1:nth-child() 和h1同一层次的其他子元素也全部参与计算
h1:nth-of-type() css筛选器首先会排除不是h1元素的影响，只专注h1元素
子元素用margin-top 父元素用overflow:hidden,子元素用浮动，父元素也用overflow:hidden

box-sizing:border-box 切换盒模型  为元素指定的任何的内边距和边框都在已设定的宽度和高度内进行设置

子元素有高度父元素没高度说明子元素浮动会造成父元素的高度塌陷，需要给浮动元素的父级用清浮动

背景图：不常更换，logo，搜索图标  img插图：常更换的(轮播图)
目标伪类：target 突出显示活动的HTML锚点
例：当你a链接到这个元素时，想要改变#tab元素的字体大小，
可以设置 #tab:target{font-size:30px}

.box1 span 后代，.box1下所有的span(不管是哪一个级别的)
.box1>span 父子，.box1下能看到的那一层span(不包括其他元素下的span)
.box1 li+li 相邻 li下边的li(相邻的li) 一般用到底部的a加左/右小竖线
.box1 ol~li 兄弟 ol下边所有的li

min-height:内容不多时也会保持最小高度，超出规定高度时会自动扩大
max-height:内容不多时会自动缩小到内容宽度，超出规定高度时不延伸(扩大)
min-width:窗口缩小到规定宽度时，会出现滚动条，平时会自动扩大到窗口大小
max-width:窗口缩小时会跟随窗口一起缩小，直到不能缩小为止
以上四个用区间理解会好一点

需要自适应父元素高度的子元素: height:100%
iconfont:购物车➡下载代码➡解压完之后重命名(iconfont)➡引入文件(见文件中的解析)
整体的宽度不够时，盒子里的内容会掉下来

文字环绕效果一般给img加浮动
a img 给img 加display:block
a中的height可以改为 line-height
background-position: right(h) 8px(v)
background-position: 背景位置(有三种写法) 水平(h) 垂直(v)
第一种：left,right,top,bottom,center
第二种：10%,50%
第三种：固定值

量盒子时不量边框
plus版是3倍解析度，所以要除以3
如果一行上有两个及以上的元素在同一行上，则需给这些元素都添加float
text-indent,text-align是块级元素，所以给行内元素会没有效果，要将行元素转为块元素
background-size:背景大小 cover:等比缩放，会全覆盖盒子，但背景图并不完全在盒子中
content:等比缩放，会显示全部，可能有留白，盖不住这个盒子，背景图最大会在盒子内

渐变：background-image: linear-gradient(to right,#xxx,#xxx);其中right是灵活的
图片的响应式：width:100% height:auto max-width:自己量的图片大小
height:inhert(继承)
